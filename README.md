# GTA Radio Privacy Policy Site

This folder is ready to upload to a public GitHub repository and publish with GitHub Pages.

Files:

- `index.html` - the privacy policy page
- `.nojekyll` - tells GitHub Pages to serve the site directly

Before publishing:

1. Open `index.html`
2. Replace `[REPLACE WITH YOUR SUPPORT EMAIL]` with your real support email

After publishing on GitHub Pages, use the public URL in Google Play Console as your Privacy Policy URL.
